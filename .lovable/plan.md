
# AI Nova Club — Premium Landing Page

## Overview
A sleek, Apple-inspired landing page for AI Nova Club — a private network for Instagram theme page owners. Black & white minimalist aesthetic with glassmorphism accents and smooth scroll animations.

## Design Direction
- **Colors**: Pure black/white base with subtle gray gradients, glassmorphic cards (frosted glass with white/10% borders)
- **Typography**: Inter font — clean, modern, San Francisco-like
- **Spacing**: Generous whitespace, large section padding
- **Animations**: Fade-in on scroll, subtle hover effects on cards/buttons, animated gradient background on hero

## Page Sections

### 1. Hero Section
- Bold headline: "Build. Monetize. Dominate."
- Subheadline about the exclusive network
- Glowing "Apply Now" CTA button (links to JotForm URL placeholder)
- Animated gradient background with subtle grain texture

### 2. Social Proof
- "Trusted by growing creators" with stats: 50+ Pages, 1M+ Reach, 500K+ Followers
- Minimal stat cards with fade-in animation

### 3. Value Proposition (4 glassmorphic cards)
- Brand Deals Access, Monetization Opportunities, Private Network, Growth Systems
- Frosted glass card style with subtle borders and hover lift

### 4. How It Works (3 steps)
- Apply → Get Approved → Start Earning & Growing
- Clean numbered steps with connecting lines

### 5. Exclusivity Section
- "Not everyone gets in" — bold statement about quality control
- Dark background section for contrast

### 6. Final CTA
- "Ready to Join?" with large centered "Apply Now" button

### 7. Footer
- Minimal: brand name, Instagram link placeholder, copyright

## Technical Details
- Single index route with all sections (single landing page)
- Intersection Observer for scroll-triggered fade-in animations
- Fully responsive mobile/desktop layout
- All "Apply Now" buttons linked to a configurable JotForm URL constant
- Dark theme by default (dark class on html)
